Gerao de relatrios com RMarkdown - Aula 4

O que ? E por que usar?

  • O markdown uma linguagem de marcao para gerao de documentos
  • Os desenvolvedores do RStudio, criaram uma verso integrada, chamada RMarkdown
  • 99% do que funciona com Markdown, funciona com o RMarkdown, mas tem aquele 1%….
  • Com o RMarkdown podemos gerar:
    • Apresentaes
    • Word
    • PDF (para artigos, recomendo o uso do Knitr, com LaTeX)
    • HTML
  • Facilidade com versionamento de cdigo (espero abordar em cursos futuros)

Criando um arquivo de exemplo

  • No RStudio, selecionem:
    • File -> New File -> R Markdown
    • Coloquem um ttulo
    • Vamos deixar marcado a opo HTML
    • Cliquem em OK
  • Compilando o arquivo com ctrl+shift+k (ou boto Knit)
  • Gerando PDF e Word

Cuidados ao colocar cdigo

  • Por ser uma linguagem bastante genrica, nem todas as funcionalidades de um tipo de documento, pode funcionar em outro
  • Por exemplo, se introduzirmos cdigo \(LaTeX\) dentro do documento, ele ser compilado em PDF normalmente, mas apresentar problemas de exibio em HTML ou Word
  • Portanto, devemos definir PARA QUE queremos gerar o nosso documento:
    • Artigo -> PDF
    • Apresentao -> HTML/PDF
    • Pgina WEB -> HTML
    • Perder tempo -> DOC

Captulos e sees

  • ## Seo (equivalente ao h2 em html)
  • ### Sub-seo (equivalente ao h3 em html)
  • O Markdown aceita at 6 nveis de formatao

Quebra de linha

  • O Markdown no quebra a linha como est no editor, a quebra feita de forma automtica
    • WYSIWYG x WYMIWYG
  • Ou, devemos deixar 2 ou mais espaos entre as linhas para mostrar que queremos quebrar a linha.

Listas

Principais parmetros do chunk de cdigo

  • Podemos passar alguns parmetros teis nos chunks:
    • eval: se FALSE, o knitr no avaliar o chunk
    • include: se FALSE, executa o chunk, mas no coloca no documento final
    • echo: se FALSE, no exibe o cdigo, apenas o resultado final
    • results: se 'asis', exibe o cdigo sem formatao de markup (por exemplo, quando queremos retornar um cdigo de tabela em \(LaTeX\))
    • error,message,warning: se FALSE, no exibe erros, mensagens ou avisos ao executar o cdigo
    • cache: se TRUE, o chunk no ser recompilado, at ser alterado

Principais parmetros do chunk de cdigo

  • Opes para figuras:
    • fig.align: 'left','right','center','default'
    • fig.cap: Ttulo da imagem, em \(LaTeX\)
    • fig.lp: prefixo do label da imagem em \(LaTeX\)
    • fig.width: largura da imagem
    • fig.height: altura da imagem

Plotando grficos com R

O que o Shiny?

Criando uma pgina com Shiny

  • Existem 2 modos principais de se criar uma aplicao com Shiny
    • 2 arquivos: ui.R e server.R
    • 1 arquivo: app.R (mais fcil de manter, na minha opinio)
  • Vamos criar nosso primeiro aplicativo com Shiny:
    • File -> New File -> Shiny Web App
    • Dem um nome para sua aplicao
    • Escolham Single file app
    • Escolham um lugar para salvar
  • No arquivo app.R que foi criado, cliquem em Run App
